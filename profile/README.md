# 🚀 NasDock

> A modern application ecosystem built for NAS.

NasDock 是一套专注于 **NAS 场景** 的应用生态，致力于为个人用户和家庭用户提供 **简单、优雅、可自托管（Self-hosted）** 的数字生活解决方案。

---

## 🧩 产品矩阵

我们正在构建一系列围绕 NAS 的核心应用：

### 🎵 AudioDock

一个现代化的音频管理与播放系统，支持本地音乐整理、在线播放与多端访问。

### 📚 BookDock

你的私人电子书管理中心，支持多格式阅读、分类整理与跨设备同步。

### ☁️ CloudDock

轻量级私有云工具，帮助你更高效地管理文件、分享资源与构建个人云空间。
